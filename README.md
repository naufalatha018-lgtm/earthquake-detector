# 🌍 Gempa Bumi Monitor (Flutter IoT Simulation)

A real-time earthquake monitoring simulation app built with Flutter.  
This project demonstrates **live data streaming, state management, charts, and alert system** with a modern UI experience.

---

## 🚀 Features

- 📊 Real-time earthquake magnitude simulation
- 🌍 Dynamic region/location updates
- 📈 Live line chart visualization (fl_chart)
- 🔊 Earthquake alarm system (audio alert)
- 🧠 State management using Provider
- 🎮 Demo mode for presentation/offline usage
- 🛑 Stop / restart simulation control
- ⚡ Smooth and responsive UI

---

## 🎯 Project Purpose

This project is designed as a **simulation of an IoT-based earthquake monitoring system**, useful for:

- School / college presentation
- IoT portfolio showcase
- UI/UX dashboard practice
- Real-time system simulation demo

---

## 🧠 Tech Stack

- Flutter 💙
- Provider (State Management)
- fl_chart (Data Visualization)
- audioplayers (Sound Alert System)
- Dart Timer (Real-time simulation logic)

---

## 📂 Project Structure

lib/
 ├── state/providers/   # App logic (GempaProvider)
 ├── presentation/      # UI screens
 ├── core/theme/        # Styling system

## 🔊 Alarm System

When earthquake magnitude reaches a threshold (≥ 5.0),  
the app automatically triggers an **emergency sound alert**.
